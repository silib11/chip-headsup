# ポーカー ヘッズアップ チップ管理

ポーカーのヘッズアップ（1対1）におけるチップ管理を行うWebアプリケーション。

## 機能

- プレイヤーA/Bのチップ数管理
- ブラインド（SB/BB）設定と途中変更
- フェーズごとのベット管理（プリフロップ、フロップ、ターン、リバー）
- ショウダウン時の自動チップ移動
- フォールド時の即座のチップ移動
- オールイン検知とショウダウンボタン強調表示

## 使い方

1. `index.html` をブラウザで開く
2. 初期チップ数とブラインドを設定
3. ゲーム開始後、各プレイヤーのベット額を入力してベット
4. フェーズを進めるか、フォールド/ショウダウンで決着
5. 勝者を入力してチップを移動

## 技術

- 単一HTMLファイル（HTML/CSS/JavaScript）
- 外部依存なし
